## What is data science?

Data science is the process of extracting insights and useful information from data using:

```txt
Programming + Mathematics + Statistics + Domain Knowledge
```

It combines technical and analytical skills to solve real-world problems.

The ultimate goal is:

> Turning raw data into useful decisions and insights.

---

## Core skills of a data scientist

### 1. Programming

Programming is required to:

- Process data
- Automate workflows
- Build models
- Analyse datasets

Primary language used in this book:

```python
Python
```

Important libraries later include:

- NumPy
- pandas
- matplotlib
- scikit-learn

---

### 2. Mathematics

Mathematics provides the foundation for:

- Linear algebra
- Optimisation
- Calculus
- Probability

Examples:

- Matrices
- Vectors
- Gradient descent
- Neural networks

---

### 3. Statistics

Statistics helps with:

- Understanding uncertainty
- Data distributions
- Hypothesis testing
- Inference

Examples:

- Mean
- Median
- Variance
- Correlation

---

### 4. Domain knowledge

Understanding the real-world problem is critical.

Examples:

| Domain | Example |
|--------|---------|
| Healthcare | Disease prediction |
| Finance | Fraud detection |
| Aviation | Delay prediction |
| E-commerce | Recommendations |

Without domain understanding, even technically correct models may fail.

---

## Why learn from scratch?

The book intentionally teaches concepts without heavily relying on libraries.

Instead of:

```python
from sklearn.linear_model import LinearRegression
```

we will eventually build many ideas ourselves.

Benefits:

- Better intuition
- Deeper understanding
- Strong debugging skills
- Better interview performance
- Strong ML foundations

---

## Important takeaways

### Data science is interdisciplinary

It combines multiple fields:

```txt
Programming
+ Mathematics
+ Statistics
+ Domain Expertise
```

No single skill is enough.

---

### Understanding > memorisation

Blindly using libraries is dangerous.

Real learning comes from:

- Implementing concepts manually
- Understanding assumptions
- Exploring failures

---

### Libraries are abstractions

Libraries simplify complex work.

However:

> If I understand what happens underneath, I can debug and reason better.

---

### The goal is problem solving

Data science is not about building fancy models.

It is about:

> Solving meaningful problems using data.
